# Steam-Game-Grabber
This application will grab steam games based off of steam id and will allow you to see how much the players have played total, how much they have played for the past 2 weeks and what games they have. If you like a certain game, you can double click that game under neath the all games tab and either open the game or visit the store page to buy the application.
